Ajuda & Referência

Este site é um flashcard interativo para estudo dos sistemas de escrita japonesa: Hiragana, Katakana, Kanji N5/N4 e Suuji (números). Use os controles abaixo para navegar pelos caracteres e praticar a leitura.

Controles & Atalhos

Tecla Ação Descrição
Space Falar Revela a leitura e reproduz o áudio do caractere
P Play · Pause Inicia ou pausa o avanço automático com timer
→ / > Próximo Avança para o próximo caractere
← / < Anterior Volta para o caractere anterior
F Fullscreen Abre o modo tela cheia para estudo focado

Como usar

Selecione as fileiras de caracteres que deseja praticar usando as caixas de seleção na parte inferior da página. Clique em Play (ou pressione Space) para ouvir a pronúncia. Use o Timer para avançar automaticamente em intervalos definidos — ideal para revisão rápida.

No modo Fullscreen, o caractere ocupa toda a tela, ótimo para treinar sem distrações ou para exibir em um monitor maior durante os estudos.
