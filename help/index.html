Help & Reference

This site is an interactive flashcard app for studying Japanese writing systems: Hiragana, Katakana, Kanji N5/N4 and Suuji (numbers). Use the controls below to navigate through characters and practise reading.

Controls & Shortcuts

Key Action Description
Space Say Reveals the reading and plays the character's audio
P Play · Pause Starts or pauses the auto-advance timer
→ / > Next Skip to the next character
← / < Previous Go back to the previous character
F Fullscreen Opens fullscreen mode for focused study

How to use

Select the character rows you want to practise using the checkboxes at the bottom of the page. Click Play (or press Space) to hear the pronunciation. Use the Timer to auto-advance at set intervals — great for quick review sessions.

In Fullscreen mode, the character fills the entire screen — ideal for distraction-free study or displaying on a larger monitor.
